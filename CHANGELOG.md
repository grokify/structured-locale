## [v0.1.0] - 2026-02-08

### Highlights

- Initial release with BCP 47 locale parsing and message translation with CLDR pluralization support

### Added

- `locale` package with BCP 47 tag parsing and normalization ([`fff2997`](https://github.com/grokify/structured-locale/commit/fff2997))
- `locale.Parse()` for parsing tags like `en-US`, `zh-Hans-CN` ([`fff2997`](https://github.com/grokify/structured-locale/commit/fff2997))
- `locale.FallbackChain()` for locale fallback sequences ([`fff2997`](https://github.com/grokify/structured-locale/commit/fff2997))
- `messages` package with translation bundle management ([`b7eadb4`](https://github.com/grokify/structured-locale/commit/b7eadb4))
- `messages.Bundle` for loading and managing translations ([`b7eadb4`](https://github.com/grokify/structured-locale/commit/b7eadb4))
- `messages.Localizer` with `T()`, `Tn()`, `Tf()` translation methods ([`b7eadb4`](https://github.com/grokify/structured-locale/commit/b7eadb4))
- CLDR plural category support (zero, one, two, few, many, other) ([`b7eadb4`](https://github.com/grokify/structured-locale/commit/b7eadb4))
- Template variable substitution with `{{.Name}}` syntax ([`b7eadb4`](https://github.com/grokify/structured-locale/commit/b7eadb4))
- Embedded default translations for 6 locales (en, de, es, fr, ja, zh) ([`b7eadb4`](https://github.com/grokify/structured-locale/commit/b7eadb4))
- JSON Schema for message files (`schema/messages-v1.schema.json`) ([`b7eadb4`](https://github.com/grokify/structured-locale/commit/b7eadb4))
